## 介绍

VibeUI 是一个简洁的前端 UI 框架,基于 vue3.0 的开源 UI 组件库, 使用`vite` `vue3.0` `typescript`搭建,同时 VibeUI 还支持`移动端`显示，该项目仅供学习。（请勿在生产环境中使用）
